## Installation

1. `sanity install simple-star-rating`

## Usage

```js
{
  title: "Rating",
  name: "rating",
  type: "simpleStarRating", // Required
  description: "Apply a rating out of 5 stars",
  options: {
    stars: 5, // Optional. Default 5.
  }
},
```
